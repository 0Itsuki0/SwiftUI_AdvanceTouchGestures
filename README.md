# SwiftUI_AdvanceTouchGestures

A demo of creating a custom touch gesture recognizer to get full information of UITouches and UIEvents, allowing us to get information such as 
- location of multiple simultaneous touches
- touch prediction
- force
- angle
- finger vs apple pencil

and etc.
